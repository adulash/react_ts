# React + TypeScript - مشروع تعليمي شامل

## فكرة المشروع

هذا المشروع هو **دليل تعليمي تفاعلي** لتعلم React مع TypeScript. يغطي المفاهيم الأساسية والمتقدمة في React من خلال أمثلة عملية وتطبيقية.

## الهدف من المشروع

تم بناء هذا المشروع كمراجعة شاملة لجميع مفاهيم React الأساسية والمتقدمة، حيث يقدم:

- **أمثلة عملية** لكل مفهوم في React
- **شرح تفاعلي** للمفاهيم المعقدة
- **تطبيق عملي** على جميع المواضيع المدروسة
- **مشروع متكامل** يوضح الفهم الكامل للموضوعات

## المواضيع المغطاة

### المفاهيم الأساسية
- **Props** - تمرير البيانات بين المكونات
- **Conditional Rendering** - التصيير الشرطي
- **Event Handling** - معالجة الأحداث
- **State** - إدارة الحالة
- **State with Objects & Arrays** - إدارة الحالة مع الكائنات والمصفوفات

### Hooks المتقدمة
- **useContext** - مشاركة البيانات بين المكونات
- **useRef** - الوصول المباشر لعناصر DOM
- **useEffect** - التعامل مع الآثار الجانبية
- **useMemo** - تحسين الأداء بالحفظ
- **useCallback** - تحسين الأداء بالدوال
- **React.memo** - منع إعادة التصيير غير الضرورية

### مفاهيم إضافية
- **Custom Hooks** - بناء Hooks مخصصة

## التقنيات المستخدمة

- **React 19** - مكتبة واجهة المستخدم
- **TypeScript** - للكتابة الآمنة للكود
- **Vite** - أداة البناء السريعة
- **React Router** - للتنقل بين الصفحات
- **Tailwind CSS** - للتصميم
- **DaisyUI** - مكونات UI جاهزة

## كيفية التشغيل

```bash
# تثبيت المكتبات
npm install

# تشغيل المشروع في وضع التطوير
npm run dev

# بناء المشروع للإنتاج
npm run build

# معاينة البناء
npm run preview
```

## هيكل المشروع

```
src/
├── components/      # المكونات التعليمية
├── pages/          # الصفحات الرئيسية
├── hooks/          # Custom Hooks
├── contexts/       # React Contexts
└── layout/         # تخطيط التطبيق
```
